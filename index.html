About Perceptron

Perceptron is an advanced AI assistant designed to seamlessly integrate various AI models, offering features like conversational AI, image recognition, object detection, and more.

Features

  • Conversational AI
  • Image Recognition
  • Sentiment Analysis (WIP)
  • Object Detection
  • Optical Character Recognition (WIP)

Manual Commands

Perceptron can perform specific tasks using simple commands, such as checking the time, weather, searching on Google or Wikipedia, and more.

How to Run

Clone the repository, install the dependencies, and run the Perceptron script. Please note that some features are still under development.
